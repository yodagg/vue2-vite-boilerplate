# vue2-vite-boilerplate
Build vue2 project with vite.
> Note: The boilerplate is still being tested

## Features
| Feature | Description |
| ------ | ----------- |
| vite | ✅ |
| vue2 | ✅ |
| vue-router | ✅ |
| vuex | ✅ |
| vue-i18n | ✅ |
| typescript | ✅ |
| setup | ✅ |
| jsx | ❌ |


## Usage
```
git clone git@github.com:yodagg/vue2-vite-boilerplate.git

cd vue2-vite-boilerplate

npm install

npm run dev
```
